# One-Shot Builder

Use this when you want a stronger first output from Claude Design and want to reduce endless iteration.

## Input checklist

- What you want built
- Exact audience
- Primary action
- Must-have sections or screens
- Style direction
- Things to avoid

## Output you want

- a fuller first-pass design
- stronger structure
- better copy assumptions
- fewer obvious mistakes

## Paste into Claude Design

```text
I want a strong first-pass design, not a generic draft.

Build:
[landing page / dashboard / onboarding / pricing page / mobile app screen]

For:
[specific audience]

Primary action:
[signup / buy / book / explore / compare]

Must include:
[sections, components, or screens]

Visual direction:
[clean SaaS / modern editorial / playful startup / premium dark / bold geometric]

Prioritize:
[clarity / trust / speed / conversion / premium feel / depth]

Avoid:
[generic hero copy / clutter / too many cards / weak CTA / template feel]

Before generating, make explicit decisions about:
1. layout hierarchy
2. section order
3. copy tone
4. visual emphasis
5. spacing density

Then generate the design based on those decisions.
```
