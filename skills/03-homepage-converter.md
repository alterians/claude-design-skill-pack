# Homepage Converter

Use this when your landing page feels generic, weak, confusing, or low-converting.

## Input checklist

- Current homepage copy or screenshot
- Offer
- Target user
- Main CTA
- Trust signals you have
- Biggest objection from users

## Output you want

- better hero structure
- stronger headline options
- improved section order
- clearer CTA logic
- friction fixes

## Paste into Claude Design

```text
Act as a conversion-focused homepage strategist.

I want you to improve this homepage so it becomes clearer, sharper, and more persuasive.

Current homepage:
[paste copy, attach screenshot, or describe structure]

Offer:
[what is being sold or promoted]

Audience:
[who it is for]

Main CTA:
[start free / book demo / buy now / join waitlist]

Proof available:
[testimonials / metrics / logos / case studies / before-after]

Main objection:
[too expensive / too complex / not trustworthy / not urgent]

Deliver:
1. Why the current page underperforms
2. A stronger hero formula
3. 5 headline options
4. A revised section order
5. Better CTA placement strategy
6. Specific copy improvements section by section
7. A final recommended homepage structure

Optimize for conversion, not awards.
```
