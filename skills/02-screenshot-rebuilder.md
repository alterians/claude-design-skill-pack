# Screenshot Rebuilder

Use this when you have a reference screenshot, website, or visual inspiration and want Claude Design to reinterpret it instead of copying it blindly.

## Input checklist

- Screenshot or URL
- What you like about it
- What you want to change
- Your brand/product context
- Screen type: landing page, dashboard, onboarding, pricing, etc.

## Output you want

- layout breakdown
- visual principles
- reusable component ideas
- adaptation plan for your own product
- clear difference from the reference

## Paste into Claude Design

```text
Analyze this reference and rebuild its logic for my product without cloning it.

Reference:
[attach screenshot or paste URL]

What I like:
[layout / hierarchy / colors / spacing / clarity / premium feel / motion]

What must change:
[brand tone / audience / offer / style / density / platform]

My product:
[describe your product briefly]

Deliver:
1. What makes this reference effective
2. The layout pattern behind it
3. The component system behind it
4. How to adapt it for my product
5. What to remove or redesign so it feels original
6. A proposed new direction for Claude Design to generate

Do not copy. Abstract the principles and reapply them.
```
