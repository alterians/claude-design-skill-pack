# Launch Content Pack

Use this when the design is finished and you want content for sharing it, explaining it, or publishing it.

## Input checklist

- Product or page summary
- Key feature or value proposition
- Target audience
- Tone
- Where you will publish

## Output you want

- demo video outline
- launch post
- changelog
- X/LinkedIn thread
- short tagline options

## Paste into Claude Design

```text
Turn this design/product into a launch-ready content pack.

Product/Page:
[describe it briefly]

Audience:
[who should care]

Main value:
[why it matters]

Tone:
[founder-led / polished / technical / playful / premium]

Channels:
[X / LinkedIn / Product Hunt / GitHub / email / video script]

Deliver:
1. 10 short tagline options
2. A 30-60 second demo video script
3. A GitHub repo description
4. A launch post for X
5. A launch post for LinkedIn
6. A short changelog entry
7. A 5-bullet talking points list for a video or voiceover

Keep it sharp, specific, and easy to publish.
```
